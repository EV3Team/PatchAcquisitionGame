## 프로젝트 소개

본 프로젝트는 LEGO EV3 로봇을 활용하여 격자 구조의 맵에서 패치를 획득하며 최고 점수를 달성하는 것을 목표로 합니다. 두 가지 과제로 구성되어 있으며 각각 최단경로 알고리즘과 자유경로 최적화 전략을 구현합니다.
